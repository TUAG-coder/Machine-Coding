Requirements Gathering:

1. User should be able to book a ride using our application.
2. The pickup location can be the user's current location or it can be selected manually by the user.
3. Both pickup and destination locations need to be given by the user before booking the ride.
4. Based on the selected source and destination, different ride options would be listed along with the fare.
5. Our application supports different types of vehicles such as: two-wheeler, cab, cab premium, cab outstation
6. Once user selects one of the listed options, booking request will be sent to the nearby drivers.
7. User should be able to cancel the ride before the ride starts.
8. There will be multiple payment options such as wallet money, cash, credit card, debit card, UPI.
9. The drivers and passengers should be able to give rating to each other.
10. For each ride a unique code will be generated which needs to be given to the driver to start the ride.
11. Once the ride is booked, driver details such as driver name, vehicle number, rating, contact number and current location will be presented to the driver.
